# rover.castalia.institute

Autonomous Field Rover — Castalia Institute

A modular robotic platform for environmental sensing, terrain exploration, and autonomous navigation in high-desert terrain.

**Live site:** https://rover.castalia.institute

## Development

```bash
npm install
npm run dev
```

## DNS

`rover.castalia.institute` is a CNAME pointing to `inquiryinstitute.github.io`, managed in Route 53.

```bash
AWS_PROFILE=custodian ./setup-route53.sh
```
